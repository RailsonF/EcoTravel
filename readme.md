**Título do Projeto:**
Eco Cariri, o turismo ecológico no Cariri.

**Descrição do estudo de caso:**

O Eco Cariri é um projeto que visa promover o turismo na região do Cariri (CE) conectando visitantes a pontos naturais, trilhas e serviços locais. O Eco Cariri visa diminuir o problema de baixa visibilidade dos atrativos locais.
